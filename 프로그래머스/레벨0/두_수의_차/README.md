# 두 수의 차

### 문제 설명

정수 `num1`과 `num2`가 주어질 때, `num1`에서 `num2`를 뺀 값을 return하도록 soltuion 함수를 완성해주세요.

-----------
### 제한 사항

- -50000 ≤ `num1` ≤ 50000
- -50000 ≤ `num2` ≤ 50000

-----------
### 입출력 예

| num1 | num2 | result |
|------|------|--------|
| 3    | 2    | 1      |
| 10   | 5    | 0      |
