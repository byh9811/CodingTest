# 머쓱이보다 키 큰 사람

### 문제 설명

머쓱이는 학교에서 키 순으로 줄을 설 때 몇 번째로 서야 하는지 궁금해졌습니다. 머쓱이네 반 친구들의 키가 담긴 정수 배열 `array`와 머쓱이의 키 `height`가 매개변수로 주어질 때, 머쓱이보다 키 큰 사람 수를 return 하도록 solution 함수를 완성해보세요.

-----------
### 제한 사항

- 1 ≤ `array`의 길이 ≤ 100
- 1 ≤ `height` ≤ 200
- 1 ≤ `array`의 원소 ≤ 200


-----------
### 입출력 예

| array                | height | result |
|----------------------|--------|--------|
| [149, 180, 192, 170] | 167    | 3      |
| [180, 120, 140]      | 190    | 0      |
