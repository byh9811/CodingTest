# 두 수의 나눗셈

### 문제 설명

정수 `num1`과 `num2`가 매개변수로 주어질 때, `num1`을 `num2`로 나눈 값에 1,000을 곱한 후 정수 부분을 return 하도록 soltuion 함수를 완성해주세요.

-----------
### 제한 사항

- 0 < `num1` ≤ 100
- 0 < `num2` ≤ 100

-----------
### 입출력 예

| num1 | num2 | result |
|------|------|--------|
| 3    | 2    | 1500   |
| 7    | 3    | 2333   |
| 1    | 16   | 62     |
